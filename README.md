# Jlpt-Go

Bonjour !
Voici un backup du site jlpt-go avant que son contenu ne soit plus disponible.

Je ne suis, bien évidemment, pas l'auteur de ce contenu. 

Pour ceux qui ne sont pas habitués au site, voici ce qu'il faut faire pour ouvrir le contenu dans un navigateur :
- téléchargez l'ensemble des fichiers (code -> download ZIP)
- Décompressez l'archive avec votre gestionnaire favori
- Ouvrez "index.html" avec votre navigateur favori
